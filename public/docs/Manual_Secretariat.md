# AFSMS v1.0 User Manual - Secretariat Role

## 1. Overview
As a member of the Secretariat, you are responsible for the core academic data of the Faculty. This includes student registration, curriculum management, and official record keeping.

## 2. Student Management
- **Manual Registration**: Navigate to the 'Students' page and click 'Enroll New Student'.
- **Bulk Import**: Use the 'Import from Excel' button to upload a spreadsheet of students. The system will automatically validate the data and provide **Smart Suggestions** if duplicates or errors are found.

## 3. Curriculum & Disciplines
- **Institutional Domains**: Define the specialization fields (e.g., CTI, IS).
- **Study Plans**: Create and activate new curricula for specific academic years.
- **Module Engineering**: Add disciplines to a curriculum, specifying credits, semester, and evaluation type.

## 4. Grade Management
- View and manage grades across all formations.
- Use the 'Import Grades' feature for mass updates.

## 5. Smart Validation
The system features an intelligent error handling engine. If you encounter a database constraint (e.g., trying to add a student ID that already exists), a blue banner will appear with a **Smart Resolution Advice** to guide you on how to fix the issue.
