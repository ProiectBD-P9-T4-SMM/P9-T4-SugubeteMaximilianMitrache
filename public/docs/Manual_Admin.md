# AFSMS v1.0 User Manual - Administrator Role

## 1. System Configuration
- **Institutional Metadata**: Set the Faculty name, Dean, and contact details.
- **Academic Years**: Manage active and historical years.

## 2. User Management
- Create accounts for staff and students.
- Assign roles (ADMIN, SECRETARIAT, PROFESSOR, STUDENT).

## 3. Data Integrity & Audit
- **Audit Logs**: Monitor every change made to the system.
- **Rollback**: In case of critical errors, administrators can initiate a database recovery.
